Web semantica

Elementos con significado!

Ejemplos de etiquetas no semánticas: Div, Span
Ejemplos de etiquetas semánticas: TAble, Form, Article

Header

Es el encabecazo de una página, sección o artículo. Incluye elementos como el logo, texto introductorio y puede tener menú de navegación. Puede haber múltiples header en una pagina. pero no puede haber un header dentro de un footer u otro header.

Footer

Es el pie de una página de un sitio o una sección. Puede haber múltiples por página. Puede incluir información como:
  • Información de contacto
  • Enlaces a las RRSS
  • Mapa del Sitio
  • Créditos
  • Formulario de Contacto

Nav

Define un conjunto de enlaces de navegación. No todos los links deben estar dentro de un "Nav"

Main

Crea un contenedor para el contenido principal de uestra página. Este debe ser único y no debe incluir contenido repetido de otras secciones, como por ejemplo del sidebar, logos, etc. No puede ser hijo de etiqutas como: article, aside, footer, header o nav.

Aside

Define el contenido relacionado indirectamente con el contenido principal, como por ejemplo un sidebar. Elementos que solemos poner en un aside:
  • Publicidad
  • Biografiad el autor
  • Enalces relacionados
  • Información del perfil

Section

Define una seccion en una página donde se agrupa los contenidos relacionados y normalmente incluyen un título
Una web se compone de varias secciones, cada una con su contenido y propósitos.
  • Capítulos
  • Introduccion
  • Noticias
  • Información de Contacto

Article

Define un contenido independiente. El mismo puede ser compartido o reutilizado en otro sitio sin requerir contexto.
Tambien suele incluir un título. Ejemplos de uso:
  • Un post d eun blog.
  • Un artículo de una revista
  • Comentario de un usuario (Reseña)
  • Tarjeta de producto

Puedo encontrar artículos dentro de secciones, y secciones dentro de artículos. Un artículo puede tener múltiples secciones. Una seccion puede tener múltiples artículos.

Figure

Define elementos autocontenidos, como puede ser una imágen, diagrama, gráfico, tabla, foto.
Dentro de ella, se suele utilixar la etiqueta "figcaption" para describir el contenido, esto es además del ya icluido alt en caso de imágenes.

Mark

Define un texto resaltado, normalmente con fondo amarillo.
